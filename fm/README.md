## 另外：

1、batch_size=128，指的是过多少个样本更新一次模型的参数，相当于每个融合矩阵就是128个样本的结果，128个样本算作一组，得到结果

2、user_id item_id算进去了，不知道为什么，但在FFM中，没有算进去，不应该算进去的！！在FM.py中做修改，加上：cols.remove('user_id')；cols.remove('item_id')
